# Anime Text Mining and Sentiment Analysis Project

## Overview
This project analyzes anime-related text data using text mining and machine learning techniques. The goal is to explore trends, sentiment, and patterns within anime datasets using Python and data analytics methods.

## Dataset
Dataset used:
- tidy_anime dataset from TidyTuesday

## Tools and Libraries
- Python
- Pandas
- NLTK
- Scikit-learn
- Matplotlib
- Altair

## Methods
- Text preprocessing
- Tokenization
- Sentiment analysis
- Word frequency analysis
- Data visualization
- Machine learning classification

- final_project_edit.ipynb
- Poster presentation materials
- Dataset references
